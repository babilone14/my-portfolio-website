About Me

ผมมีความมุ่งมั่นที่จะเป็น Software Engineer เพราะตลอดเวลาที่ผ่านมาผมได้ลงมือสร้างผลงานจริงทั้งแอปพลิเคชั่นอย่างง่าย My School Planner รวมถึงการทำ portfolio website นี้ที่ช่วยพัฒนาทักษะการเขียนโค้ด การแก้ปัญหา และการคิดสร้างสรรค์ ประสบการณ์เหล่านี้สะท้อนให้เห็นถึงความตั้งใจในการเรียนรู้และพัฒนาตัวผมเองอย่างต่อเนื่อง

สำหรับเป้าหมายในอนาคตของผม ผมจะต่อยอดทักษะทั้งด้านการพัฒนา Software เต็มรูปแบบ การทำงานกับ Data และ AI เพื่อสร้างเทคโนโลยีหรือนวัตกรรมที่มีประโยชน์ต่อสังคมรวมถึงยกระดับชีวิตของผู้คนทั่วไป ทั้งหมดนี้ทำให้ผมอยากเป็น Software Engineer นั่นเองครับ

Data Analysis

ผมมีประสบการณ์วิเคราะห์ข้อมูล เช่น การหาปัจจัยที่มีผลต่อเงินเดือนแรกเข้า ซึ่งแสดงถึงการใช้เหตุผลเชิงตรรกะ การจัดการข้อมูล และการตีความเชิงสถิติ ทักษะนี้ทำให้ผมสามารถพัฒนาต่อไปสู่สาย Data Analytics, Machine Learning หรือ AI Engineering ได้ครับ

Excel

Coding

ผมมีทักษะการเขียนโปรแกรมด้วยภาษา C, C#, Python และ JavaScript โดยเคยใช้ในการแก้โจทย์การแข่งขัน และการทำโปรเจ็กต์ส่วนตัว ตัวอย่างเช่นการสร้างโปรแกรมคำนวดเกรดด้วย python สิ่งนี้ช่วยพัฒนาทักษะการคิดเชิงตรรกะ การแก้ปัญหาอย่างเป็นระบบ และการออกแบบโค้ดที่สามารถนำไปต่อยอดสู่การเรียนรู้ด้าน Software Development และ Data Science ได้ครับ

AI Study Buddy

ผมอยากลองเรียนรู้เกี่ยวกับการทำงานของ AI และ Natural Language Processing โดยตั้งใจพัฒนาระบบผู้ช่วยการเรียนรู้ที่สรุปเนื้อหาจาก PDF หรือโน้ตเรียนอัตโนมัติ ระบบจะประมวลผลข้อความและสกัดสาระสำคัญออกมาให้ง่ายต่อการเข้าใจ เช่น flashcard สำหรับทบทวน หรือ quiz เพื่อทดสอบความเข้าใจทันที

ผมวางแผนว่าจะใช้เทคนิคด้าน NLP และ machine learning มาช่วยในการตัดสินใจว่าเนื้อหาส่วนไหนสำคัญ ควรสรุปเป็นข้อความสั้น ๆ หรือควรถูกแปลงเป็นคำถามแบไหน เพื่อให้ผู้ใช้ได้รับสาระสำคัญที่ตรงประเด็น ไม่ยาวจนเกินไปแต่ยังคงครบถ้วนต่อความเข้าใจ ระบบนี้จะช่วยให้นักเรียนหรือผู้ใช้ทั่วไปไม่ต้องเสียเวลาอ่านเอกสารยาว ๆ ซ้ำไปมา แต่สามารถโฟกัสที่จุดสำคัญและทบทวนได้อย่างมีประสิทธิภาพ ทำให้การเรียนรู้มีความกระชับ เร็วขึ้น และตอบโจทย์ความต้องการของผู้เรียนในยุคดิจิทัล

My School Planner 2.0

จากที่ผมได้สร้างแอป My School Planner ผมตั้งใจพัฒนาต่อยอดด้วย AI เพื่อวิเคราะห์พฤติกรรมการเรียนรู้และการใช้เวลา ระบบจะเก็บข้อมูลกิจกรรม เช่น การบ้าน การสอบ ช่วงโมงเรียน และเวลาพักผ่อน เพื่อนำมาวิเคราะห์หาความสัมพันธ์ระหว่างเวลาและประสิทธิภาพการเรียน แล้วสร้างคำแนะนำตารางที่เหมาะสมที่สุด เป้าหมายคือช่วยผู้ใช้รักษาสมดุลระหว่างการเรียนกับการพักผ่อนความเครียด และเพิ่มประสิทธิภาพการเรียนรู้
