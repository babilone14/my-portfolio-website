About Me

ผมมีความมุ่งมั่นที่จะเป็น Software Engineer เพราะตลอดเวลาที่ผ่านมาผมได้ลงมือสร้างผลงานจริงทั้งแอปพลิเคชั่นอย่างง่าย My School Planner รวมถึงการทำ portfolio website นี้ที่ช่วยพัฒนาทักษะการเขียนโค้ด การแก้ปัญหา และการคิดสร้างสรรค์ ประสบการณ์เหล่านี้สะท้อนให้เห็นถึงความตั้งใจในการเรียนรู้และพัฒนาตัวผมเองอย่างต่อเนื่อง

สำหรับเป้าหมายในอนาคตของผม ผมจะต่อยอดทักษะทั้งด้านการพัฒนา Software เต็มรูปแบบ การทำงานกับ Data และ AI เพื่อสร้างเทคโนโลยีหรือนวัตกรรมที่มีประโยชน์ต่อสังคมรวมถึงยกระดับชีวิตของผู้คนทั่วไป ทั้งหมดนี้ทำให้ผมอยากเป็น Software Engineer นั่นเองครับ

Web Development

ผมมีทักษะในการสร้างเว็บไซต์พื้นฐานโดยใช้ HTML, CSS และ JavaScript ผ่านการทำ Portfolio Website ที่แสดงถึงความสามารถในการจัดโครงสร้างหน้าเว็บ ออกแบบการแสดงผล และเพิ่มฟังก์ชันการทำงานเชิงโต้ตอบ ซึ่งถือเป็นพื้นฐานสำคัญในการต่อยอดสู่การพัฒนาเว็บขั้นสูงหรือ Full-stack Development นั่นเองครับ

Mobile Development

Data Analytics

ผมมีประสบการณ์วิเคราะห์ข้อมูล เช่น การหาปัจจัยที่มีผลต่อเงินเดือนแรกเข้า ซึ่งแสดงถึงการใช้เหตุผลเชิงตรรกะ การจัดการข้อมูล และการตีความเชิงสถิติ ทักษะนี้ทำให้ผมสามารถพัฒนาต่อไปสู่สาย Data Analytics, Machine Learning หรือ AI Engineering ได้ครับ

Excel

ผมมีทักษะการใช้ Microsoft Excel ในการจัดการและวิเคราะห์ข้อมูล โดยใช้สูตร ฟังก์ชัน และการสร้าง Pivot Table เพื่อสรุปผลที่ซับซ้อนให้อ่านง่ายมากขึ้น เคยใช้ Excel ในการวิเคราะห์ข้อมูลจากโครงการวิจัย เช่น การหาปัจจัยที่มีผลต่อผลลัพธ์บางอย่าง และการสร้างกราฟสรุปข้อมูล ซึ่งเป็นประโยชน์ต่อการสื่อสารเชิงข้อมูล และยังช่วยให้ผมเข้าใจหลักการ Data Analysis เบื้องต้นมากขึ้นด้วยครับ

Programming Languages

ผมมีทักษะการเขียนโปรแกรมด้วยภาษา C, C#, Python และ JavaScript โดยเคยใช้ในการแก้โจทย์การแข่งขัน และการทำโปรเจกต์ส่วนตัว ตัวอย่างเช่นการสร้างโปรแกรมคำนวดเกรดด้วย python สิ่งนี้ช่วยพัฒนาทักษะการคิดเชิงตรรกะ การแก้ปัญหาอย่างเป็นระบบ และการออกแบบโค้ดที่สามารถนำไปต่อยอดสู่การเรียนรู้ด้าน Software Development และ Data Science ได้ครับ

Machine Learning

ผมมีประสบการณ์ในการใช้ Machine Learning โดยเฉพาะ Linear Regression ในการวิเคราะห์และทำนายผลลัพธ์ จากโครงงาน Analysis of Factors Affecting Starting Salary ที่ผมใช้ Python บน Google Colab ในการสร้างโมเดล เพื่อหาปัจจัยที่มีผลต่อเงินเดือนแรกเข้า ผมได้เรียนรู้กระบวนการตั้งแต่การเตรียมข้อมูล การวิเคราะห์เชิงสถิติ การสร้างและประเมินโมเดล รวมถึงการ Visualization ข้อมูล ทักษะนี้เป็นพื้นฐานสำคัญในการต่อยอดไปสู่ Data Science และ AI Engineering ในอนาคตครับ

Computer Vision

ผมมีความเชี่ยวชาญด้าน Computer Vision ผ่านการพัฒนาโปรเจค Virtual Mouse และ Sign Language Detection โดยใช้ Python ร่วมกับ OpenCV และ MediaPipe ในการประมวลผลภาพ ตรวจจับท่าทางมือ และควบคุมเคอร์เซอร์ ผมได้เรียนรู้การเทรนโมเดลให้สามารถแปลภาษามือเป็นข้อความได้ ซึ่งแสดงถึงความเข้าใจในการประมวลผลภาพ การตรวจจับวัตถุ และการประยุกต์ใช้ Deep Learning ทักษะนี้ทำให้ผมพร้อมที่จะพัฒนาแอปพลิเคชันที่ใช้ Computer Vision ในการแก้ปัญหาจริงและสร้างนวัตกรรมที่เป็นประโยชน์ต่อสังคมครับ

โครงงานนี้ศึกษาความเป็นไปได้ในการนำ AI มาใช้ในกระบวนการยุติธรรม เพื่อวิเคราะห์ข้อดี ข้อจำกัด และผลกระทบต่อระบบตุลาการ โดยเก็บข้อมูลจากนักเรียนมัธยมปลาย 46 คน และนิติกร 1 ท่าน พบว่าผู้ตอบส่วนใหญ่เห็นว่า AI สามารถช่วยสนับสนุนการทำงานของศาลได้อย่างมีประสิทธิภาพ ทั้งในด้านการจัดการข้อมูล การวิเคราะห์ และการตัดสินคดีบางประเภท จากการทำโครงงานนี้ ผมได้เรียนรู้กระบวนการวิจัย ข้อมูลเบื้องต้นเกี่ยวกับปัญญาประดิษฐ์ การทำงานเป็นทีม และการนำเสนอผลงานต่อคณะกรรมการผู้เชี่ยวชาญครัย

Virtual Mouse

ผมได้ศึกษาเรื่อง Computer Vision และสนใจโปรเจกต์ Virtual Mouse เพราะเห็นว่าเทคโนโลยีนี้สามารถนำไปต่อยอดและประยุกต์ใช้ได้ในหลายด้านจึงทดลองพัฒนาขึ้นมาครับ โดยใช้ Python ร่วมกับ Library ต่าง ๆ คือ OpenCV, MediaPipe, pyautogui และ pynput เพื่อสร้างระบบควบคุมเคอร์เซอร์และจำลองการคลิกเมาส์ด้วยท่าทางมือผ่านกล้องโน้ตบุ๊กได้ครับ ผมได้เรียนรู้การประมวลผลภาพด้วย OpenCV การตรวจจับข้อต่อมือด้วย MediaPipe และได้ประยุกต์โค้ดและออกแบบท่าทางให้ใช้งานได้ง่ายขึ้น ผมมีแผนจะพัฒนาโปรเจกต์นี้ต่อให้สามารถใช้เป็นแหล่งเรียนรู้ภาษามือสำหรับทุกคนได้ในอนาคตอีกด้วยครับ

Sign Language Detection

จากโปรเจกต์ Sign Language Detection ทำให้ผมต้องการที่จะพัฒนาโปรแกรมนี้ให้เป็นแพลตฟอร์มที่สอนภาษามือให้กับผู้ที่สนใจ ผมจะทำสื่อเป็นรูปภาพท่าทางภาษามือเพื่อแสดงถึงข้อความต่าง ๆ และโปรแกรมแสดงหน้าจอเพื่อให้ผู้ใช้ทดสอบท่าทางต่าง ๆ โดยจะทำระบบเป็นด่านทดสอบเป็นแต่ละบททดสอบ เช่น Greeting, Introducing yourself, Alphabet เป็นต้น รวมถึงทำระบบ log in และติดตามผลความคืบหน้าของการฝึกภาษามืออีกด้วย

สิ่งที่ผมจะเรียนรู้เพิ่มเติม : [ React, Flask, Firebase, JWT, bcrypt, Gamification ]
